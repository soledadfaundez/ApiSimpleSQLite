# Ejercicio Soledad Faúndez C, api simple CRUD con SQLite.

## Este ejercicio contempla:
- Un endpoint CRUD para Customer.
- Un endpoint para Register y Login
- Automaper
- JWT
- Login
- Validaciones
- Registro de datos en SQLite
- Swagger
- Claims
- Loggin

## Pasos para ejecutar:
- 1.- Actualizar paquetes
	PM> dotnet restore
- 2.-Compilar:
	PM>dotnet build
- 3.-Ejecutar:
	PM>dotnet run
- 4.-Revisar End-Point swagger:
	http://localhost:4200/swagger/index.html
- 5.-Realizar pruebas con Postman:
	Abrir: ApiCustomerSqlite.postman_collection.json
